# Contexts (or Store)

This directory is for global state management using React Context API, or for state management libraries like Redux (reducers, actions, store) or Zustand (stores).

Examples (for Context API):
- AuthContext.jsx
- ThemeContext.jsx
- TaskContext.jsx

Examples (for Redux):
- store.js
- reducers/
- actions/

Examples (for Zustand):
- useAuthStore.js
- useTaskStore.js
